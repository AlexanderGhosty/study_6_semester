# Лабораторная работа 2

Поиск экстремума функции на интервале `[A, B]` с помощью генетического алгоритма и визуализацией поколений в `PyQt6`.

## Запуск (через venv)

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

## Что реализовано

- Ввод `A`, `B`, `d`, типа оптимума (минимум/максимум), числа популяций.
- Ввод/изменение параметров ГА:
  - размер популяции;
  - вероятность `cut-point`;
  - коэффициенты скрещивания (`rate`, `alpha`);
  - коэффициент и сила мутации.
- Отрисовка функции на интервале `[A-d, B+d]`.
- Отрисовка особей для каждого поколения:
  - зеленый: наиболее жизнеспособная особь;
  - красный: наименее жизнеспособная особь.
- Слайдер для просмотра поколений после расчета.

## Функции для исследования

- `2*x*sin(x)-x/5`
- `3*x*sin(x)+x/3`
- `2*x*cos(x)+x/5`
